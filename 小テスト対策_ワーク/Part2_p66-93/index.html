問題概要

この課題では、あらかじめ用意された index.html(plactice.html) に マークアップされている HTML 要素に対して、 CSS を用いて装飾を行います。
完成イメージを参考にcssを適用してください。

HTML ファイルは 編集禁止 です。
新規に作成する style.css ファイルに、 指示されたスタイルを正しく記述してください。

以降の手順では、
「どの HTML 要素に」「どのような CSS を指定するか」を 手順ごとに示しています。

※ 指定されていないプロパティや数値を追加しないでください。

手順1：body

  • 背景色を #f4f6ff
  • フォントを
    Arial, Hiragino Kaku Gothic ProN, Meiryo, sans-serif の順で指定

手順2：h1

  • 文字サイズ：22px
  • 枠線：太さは2px、 種類は破線、枠線の色は#6666cc
  • 角丸：8px
  • 幅：360px
  • 余白と位置合わせ：上下に30pxの余白を設けて中央寄せ
  • 内側余白：上下 12px

手順3：ul

  • 幅：360px
  • 余白と位置合わせ：上下の余白は0、左右は中央寄せ
  • 内側余白：0
  • テキスト配置：左寄せ

手順4：li

  • 下方向の余白：12px
  • リストマーカーを非表示

手順5：a

  • 背景色：#4a90e2
  • display: block
  • 内側余白：上下 12px、左右は0
  • 角丸：10px
  • 文字色：#ffffff、太字

手順6：img

  • 角丸：12px
  • display: block
  • 余白と位置合わせ：上は10px／下は30px左右は中央寄せ

手順7：p, ul（複数指定）

  • 下方向の余白：30px
  • テキスト配置：左寄せ

手順8：全体確認

すべての指定が正しく反映されているか確認してください。

手順9：提出前チェック

  • HTMLを編集していない
  • px を省略していない
  • セレクタの指定ミスがない
